# AWS IAM Anomaly Detection System

## Overview

Developed a cloud-based anomaly detection system that monitors AWS IAM activity using CloudTrail logs and identifies suspicious behavior through statistical analysis.

## Technologies Used

- AWS Lambda
- AWS S3
- AWS DynamoDB
- AWS CloudTrail
- Python

## Key Features

- Automated CloudTrail log ingestion
- Detection of anomalous IAM activity
- Serverless architecture
- Event-driven processing
- Persistent storage of findings

## Skills Demonstrated

- AWS cloud security
- Identity and Access Management (IAM)
- Serverless computing
- Security monitoring
- Log analysis
