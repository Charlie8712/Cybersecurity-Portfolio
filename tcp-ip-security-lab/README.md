# TCP/IP Security Attack and Defense Lab

## Overview

Performed hands-on analysis of common TCP/IP vulnerabilities within a controlled Docker-based SEED Ubuntu environment. Simulated attacks and evaluated mitigation techniques.

## Technologies Used

- SEED Labs
- Docker
- Ubuntu Linux
- Wireshark
- TCP/IP Networking

## Attacks Investigated

- SYN Flooding
- TCP RST Injection
- TCP Session Hijacking

## Defense Mechanisms

- SYN Cookies
- Network Traffic Monitoring
- Packet Analysis

## Skills Demonstrated

- Network security
- Attack simulation
- Threat analysis
- Packet inspection
- Linux administration
