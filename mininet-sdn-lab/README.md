# Linux Network Emulation and SDN Lab

## Overview

Built and analyzed software-defined network topologies using Mininet and MiniEdit within a Linux virtual environment. Simulated hosts, switches, and network communication while validating connectivity and observing packet behavior.

## Technologies Used

- Linux
- Mininet
- MiniEdit
- VirtualBox
- Wireshark
- TCP/IP

## Key Activities

- Created virtual SDN topologies
- Configured hosts and switches
- Tested connectivity using networking utilities
- Captured and analyzed traffic
- Investigated packet flow behavior

## Skills Demonstrated

- Network troubleshooting
- TCP/IP analysis
- Linux administration
- SDN concepts
- Packet inspection
